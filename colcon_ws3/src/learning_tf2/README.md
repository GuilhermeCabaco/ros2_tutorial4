# The learning_tf2 package

Package to illustrate tf2. Simplified from ROS 2 tutorials demo [Introducing tf2](https://docs.ros.org/en/humble/Tutorials/Intermediate/Tf2/Introduction-To-Tf2.html).

Used in the [Tools tutorial](https://sir.upc.edu/projects/ros2tutorials/3-tools/index.html)  of the Introduction to ROS course.

## Usage

```
$ros2 launch learning_tf2 turtle_tf2_demo2.launch.py
```
